# Data Format

## Package Data Format

```json
{
    "id": 0,
    "name": "xxx",
    "cards": [
    ]
}
```

## Card Data Format

Cards are in packages.

```json
{
    "id": 0,
    "name": "xxx",
    "package": 0,
    "rarity": 0,
    "class": 0,
    "type": 0,
    "CAH": [4, 4, 5],
    "skills": [
    ]
}
```
